Comprendre les principes fondamentaux pour maintenir un environnement sain pour nos chats

Le Biofilm

Les microbes créent une matrice protectrice appelée "biofilm" qui les protège des désinfectants. C'est pourquoi :

  • Le nettoyage mécanique est INDISPENSABLE
  • Seule une brosse + détergent peut rompre le biofilm
  • La désinfection seule est INEFFICACE
  • Un bon nettoyage élimine 70-90% des microbes

Nettoyage vs Désinfection

Ces deux étapes sont distinctes et complémentaires :

  • Nettoyage : Retire les salissures et le biofilm
  • Désinfection : Tue les microbes restants
  • TOUJOURS nettoyer AVANT de désinfecter
  • Jamais de produit 2-en-1

Équipements de Protection (EPI)

Les EPI sont OBLIGATOIRES lors de la manipulation des produits :

  • Gants de protection
  • Lunettes de sécurité
  • Tablier imperméable
  • Chaussures fermées

Le Principe TACT

4 paramètres essentiels pour un nettoyage efficace

Température

Eau à la bonne température selon le produit (20-40°C)

Action

Brosser énergiquement pour casser le biofilm

Concentration

Respecter la dilution indiquée sur la fiche technique

Temps

Laisser agir le produit pendant le temps requis
